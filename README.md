# CASAWAVE – Full-Stack E-commerce Website

CASAWAVE is a modern full-stack e-commerce web application built for a clothing brand.  
It provides a seamless shopping experience for users and a powerful admin dashboard for managing products and orders.

---

## Features

### User Side
- Browse products بسهولة
- View detailed product pages
- Select sizes before ordering
- Place orders with email & phone
- Track orders بسهولة

### Admin Dashboard
- Secure authentication (JWT)
- Add / Remove products
- View all orders
- Archive orders
- Update homepage hero image

---

## Tech Stack

### Frontend
- React (Vite)
- React Router
- Axios
- CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB
- Mongoose

### Authentication & Security
- JWT (JSON Web Token)
- bcrypt
